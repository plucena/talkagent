ThoughtTreasure system requirements

Recommended configuration

Red Hat Linux 5.2 or higher
486 processor or higher
32 MB RAM or more
40 MB available disk space or more recommended (8 MB for the distribution + 32 MB for swapping)
16 MB disk space required for TTKB (optional)

Other configurations

Windows 95/98
486 processor or higher
32 MB RAM or more
40 MB available disk space or more recommended (8 MB for the distribution + 32 MB for swapping)
16 MB disk space required for TTKB (optional)
To compile ThoughtTreasure, you will need to download a copy of DJGPP version 2.01 or higher.
The ThoughtTreasure server may not run on Windows due to problems in the available socket libraries.

Solaris 2.4 x86
Pentium processor or higher
32 MB RAM or more
40 MB available disk space or more recommended (8 MB for the distribution + 32 MB for swapping)
16 MB disk space required for TTKB (optional)
You will need to compile ThoughtTreasure using the Sun cc compiler (SunSoft WorkShop, ProCompiler C).
Earlier versions ran under Solaris, but the latest version has not been tested there.

Other
ThoughtTreasure may also compile and run on other processors such as SPARC and on other operating systems such as DOS, SunOS, NextStep, FreeBSD, and NetBSD but these have not been tested.
