;_______________________________________________________________________________
; Examples:
; rhs_word0:     "poetesses"                "diskettes"
; ht:            EnglishIndex               EnglishIndex
; affix_word:    "ess"                      "ette"
; affix_obj:     N("suffix-N-to-N-female")  N("small")
; lhs_obj:       N("poet")                  N("disk")
; lhs_word:      "poet"                     "disk"
; lhs_feature:   "Nz"                       "Nz"
; lhs_pos_const: F_NOUN                     F_NOUN
; rhs_le:        poetess.Nz/                diskette.Nz/
; rhs_word:      "poetess"                  "diskette"
; rhs_obj:       N("poetess")               N("diskette")
; rule_obj:      N("suffix-N-to-N-female")  N("affix-attribute")
; lhs_class:     N("animal")                N("r1")->N("physical-object")
; lhs_pos_ptn:   F_NOUN                     F_NOUN
; rhs_class:     NULL->N("poet")            NULL->N("disk")
; rhs_feat:      ""                         ""
;
; Affixes are defined using the following relations, in terms of the above:
; lhs-class-of (ISAP check)
; -------------------------
; if NULL, match anything
; else if N("r1"), use selectional restriction on <affix_obj>
; else use the specified class
;
; lhs-pos-of
; ----------
; if NULL, match anything
; else <lhs_pos_const> must equal specified feature
;
; rhs-class-of
; ------------
; if NULL, create <rhs_obj> having <lhs_obj> as parent
; else if N("rhs-class-synonym"), <rhs_obj> is <lhs_obj>
; else create <rhs_obj> having the specified class as parent
;
; rhs-pos-of
; ----------
; if NULL, use <lhs_pos_const>
;
; rhs-assertion-of
; ----------------
; may refer to N("affix-obj") N("lhs-obj") N("rhs-obj")
;_______________________________________________________________________________
; Certain nuances of meaning aren't captured in the below (for example,
; Î  is a catch-all).
;_______________________________________________________________________________
; todo: Affixes for forming adjectives/adverbs from adjectives/adverbs are
; stored as synonyms for intensifiers. Example: hyper-."
; See (Lyons, 1977) for meanings of un- and non-.

=====nominal-relative-pronoun/nominal-pronoun/nominal#A relative#A pronoun*.z/
pronom* relatif*A nominal*A.My/
;_______________________________________________________________________________
; Nominal relative pronouns are pronouns in which a wh-element is merged with
; its antecedent: I took [what] they offered me.
;        is like: I took [the thing that] they offered me.
; We extend this class to phrases to handle such things as French "de quoi".
; Unlike "ce dont", this cannot be always be analyzed properly via its
; components: "de quoi" is really "ce de quoi".
; See Greenbaum and Quirk (1990, sections 15.7, 15.8)
; and (Grevisse, 1986, section 678).
; See also (Alexander, 1988, sections 1.24.2 and 15.20)
;_______________________________________________________________________________

; Incorporates spatial predicates and lexical entries from Dorr's (1995)
; LCS and Saint-Dizier's (1995) LCS.

; is useful. Otherwise, a single preposition concept here may actually have
; several meanings.

;   the meaning (e.g. the various meanings of 'by').

;   not yet seem to be useful to break these out into separate concepts.
; - The various interpretations of prep-of are hardcoded.
; In order to go to a one-preposition-meaning one-preposition-concept scheme,
; the preposition-of scheme for relative pronouns would need to be modified.
; (Ideally, such pronouns should refer to lexical entries and not concepts.)
